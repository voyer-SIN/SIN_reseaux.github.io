Table ASCII étendue

La table ASCII (American Standard Code for Information Interchange) étendue est une table de 8 bits. Elle contient les caractères par défaut, utilisées sur des machines ASCII natives comme Windows, Unix, VMS/VAX, etc...

Elle contient les caractères de valeurs décimales de 128 au 255, ce qui correspond aux valeurs de 10000000 à 11111111 binaire.

La table ASCII étendue, ou table 8 bits, inclue les caractères speciaux. Les caractères dans la partie étendue varient en fonction du pays; la table contient donc les caractères spécifiques qui correspondent à chaque pays, comme par exemple les caractères é, ù, è, à etc... utilisés en France.
