Table ASCII

La table ASCII (American Standard Code for Information Interchange) est une table de 7 bits. Elle contient les caractères par défaut, utilisées sur des machines ASCII natives comme Windows, Unix, VMS/VAX, etc...

Elle contient aussi les valeurs décimales de 0 à 127, ce qui correspond au 00000000 à 01111111 binaire.

La table ASCII étendue, ou table 8 bits, inclue les caractères décimaux de 0 à 127 ainsi que les caractères spéciaux, de 128 à 255. Les caractères dans la partie étendue varient en fonction du pays; la table contient donc les caractères spécifiques qui correspondent à chaque pays, comme par exemple les caractères é, ù, è, à etc... utilisés en France.

Les premiers 32 caractères sont les caractères de contrôle ou des caractères non-imprimables
